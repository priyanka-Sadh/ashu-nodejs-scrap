# ashu-nodejs-scrap

1. Go the the root folder on Terminal.
2. run `npm install`
3. run nodemon
4. Open postman
make JSON post request as below.
URL : localhost:4000/
raw json :
{
	"reqType":"aboutTextScrap",
	"keyword" : "mindcrew about"
}
